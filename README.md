# 📊 National Accounts Dashboard

A **web-based interactive dashboard** for visualizing and analyzing national economic indicators such as **GDP, GNI, Inflation, Unemployment, and Per Capita Income** across multiple countries.

---

## 🚀 Live Demo

👉 [https://nationalaccountsdashboard.netlify.app/](https://nationalaccountsdashboard.netlify.app/)

---

## 📌 Project Overview

The **National Accounts Dashboard** transforms complex economic data into **interactive visual insights**.
It enables users to:

* Explore country-level economic indicators
* Compare economies side-by-side
* Analyze trends using dynamic charts
* Download visual data for reporting

This project focuses on **data visualization, user experience, and real-time insights**.

---

## 🎯 Features

### 🌍 Country Selection

* Select any country from dropdown
* Dashboard updates dynamically based on selection

### 📊 Key Economic Indicators

* GDP Growth
* Nominal GDP
* Inflation Rate
* Unemployment Rate
* Per Capita Income
* Year-over-year comparison indicators

### 📈 Interactive Charts

* Line charts (GDP trends, growth)
* Bar charts (rankings, inflation)
* Pie charts (GDP composition)
* Smooth animations and dynamic updates

### 📑 Insight Generation

* Auto-generated summary explaining economic condition
* Highlights trends like growth, inflation, and labor market

### 🌐 Global Rankings

* Top economies visualized using bar charts
* Detailed ranking table with:

  * GDP
  * Growth rate
  * Per capita income

### 📉 Trend Analysis

* GDP trajectory over time
* Inflation and unemployment trends
* Per capita income growth

### 🧩 GDP Composition

* Breakdown of GDP into:

  * Consumption
  * Investment
  * Government spending
  * Net exports
* Sector-wise contribution:

  * Agriculture
  * Industry
  * Services

### ⚖️ Country Comparison

* Compare two countries side-by-side
* Metrics compared:

  * GDP
  * Growth
  * Inflation
  * Unemployment
  * Per capita income
* Includes auto-generated comparison insights

### ⬇️ Export Feature

* Download charts as **PNG images**
* Useful for reports and presentations

### 📱 Responsive Design

* Works on desktop, tablet, and mobile
* Clean and modern UI

## ⚙️ How It Works

1. User selects a country or comparison option
2. Frontend sends request to backend
3. Backend fetches data from World Bank API
4. Data is processed and returned as JSON
5. Frontend dynamically renders charts using Chart.js
6. UI updates instantly without page reload
